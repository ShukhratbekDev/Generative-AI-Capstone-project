## Projects

### 1. Chat with Data (Capstone Project 1)

**Location**: `chat_with_data/`

An AI-powered Data Insights Application that allows users to query and analyze sales data using natural language. Features include:

- AI Agent with function calling capabilities
- Safe database querying (SELECT only)
- Business dashboard with charts and metrics
- Support ticket creation via GitHub Issues
- Console logging for all operations

See [chat_with_data/README.md](chat_with_data/README.md) for detailed documentation.

### 2. Voice to Image (Capstone Project 2)

**Location**: `voice_to_image/`

Transform your voice into stunning AI-generated images. Simply describe what you want to see, and the app creates beautiful images using DALL-E 3. Features include:

- Voice recording and transcription
- AI-powered description enhancement
- Image generation with DALL-E 3
- Full transparency of intermediate steps
- Console logging for all operations
